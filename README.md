# Invoice-project

It can generate bills automatically.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

This repository contains the source for the invoice generator application. Follow the steps above to run it locally.
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
